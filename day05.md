# Day 5 - Processes and Jobs

## Commands Learned
- `ps aux`, `top`, `htop` -> list processes
- `jobs`, `fg`, `bg`, `CTRL-Z` -> manage jobs
- `kill <PID>` -> stop process
- `free -h~, `df -h`, `du -sh * ` -> monitor resources

## Reflections
Learned how to view, manage, and kill processes. Practiced foreground/background jobs. Useful for troubleshooting system performance.
