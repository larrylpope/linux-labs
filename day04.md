# Day 4 - Users, Groups, and Permissions (vim)

## Commands Learned
- `whoami`, `id` -> check user info
- `adduser`, `usermod`, `groups` -> manage users/groups
- `su - username` -> switch user
- `chmod`, `chown` -> change permissions
- Numeric chmod notation (e.g., 755)

## Reflections
Practiced creating users, assigning groups, switching users, and modifying file permissions. Used vim for editing and documenting notes.
