# Day 6 - Package Management

## Commands Learned
- `apt update` -> refresh package index
- `apt install <pkg>` -> install
- `apt remove <pkg>` -> uninstall
- `apt search <pkg> -> search

## Reflections
Practiced installing, removing, and searching for packages. Confirmed installation by running programs like htop. Installing additional utilities like curl, wget, tree helps with troubleshooting.
