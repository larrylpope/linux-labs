# Day 7 - System Services with systemctl

## Commands Learned
- `systemctl list-units --type=service --al` -> list services
- `systemctl status <service>` -> check service state
- `systemctl start/stop/restart <service>` -> control service
- `systemctl enable/disable <service>` -> manage boot behavior

## Reflections
Learned how to check and control system services with systemctl. Practiced starting/stopping apache2 and enabling/disabling it at boot.
