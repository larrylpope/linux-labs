# Day 8 - Logs & Monitoring

## Commands Learned
- `journalctl`, `journalctl -f` -> view structured logs
- `journalctl -u <service>` -> view logs per service
- `/var/log/syslog`, `/var/log/auth.log`, `dmesg` -> traditional log files
- `logrotate` -> manages log rotation

## Reflections
Learned how to check logs using journalctl and traditional log files. Practiced filtering by time and service (apache2). Explored system log rotation.
