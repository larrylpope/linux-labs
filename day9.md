# Day 9 - Networking Basics

## Commands Learned
- `ip addr show`, `ip route show` -> network interfaces and routing
- `ping`, `curl` -> test connectivity and HTTP response
- `ss -tuln` -> check open/listening ports
- `dig`, `nslookup` -> DNS testing
- `hostnamectl` -> system hostname and info

## Reflections
Practiced verifying connectivity, inspecting network interfaces, and checking open ports. Learned how to confirm DNS resolution and service reachability.
