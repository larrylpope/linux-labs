# Day 10 - Archiving & File Transfers

## Commands Learned
- `tar -cvf`, `tar -czvf`, `tar -xvf` -> create and extract archives
- `gzip`, `bzip2` -> compression formats
- `scp` -> secure copy over ssh
- `rsync -av` -> efficient directory sync

## Reflections
Practiced creating, compressing, and extracting archives with tar. Learned how to transfer files using scp and rsync. rsync's incremental copy is useful for backups and deployments.
