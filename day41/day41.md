# Day 41 - Git Fundamentals Review

## Objective
Practice core Git operations: clone, add, commit, push, pull.

## Commands Used
```bash
git clone https://github.com/larrylpope/linux-labs.git
git add day41.md
git commit -m "Day 41: Git Fundamentals practice file"
git push origin main
git pull origin main
```

## Verification
- File appears on GitHub
- `git log` shows commit
- Pull tested
