# Day 13 - File Operations & Automation

## Scripts
- `backup_files.sh` -- creates timestamped backups of directories
- `rotate_logs.sh` -- simulates log rotation when file size exceeds threshold
- `rename_txt.sh` -- batch renames .txt files in a directory

## Key Concepts
- File manipulation via Bash (`cp`, `mv`, `stat`, loops)
- Dynamic directory naming using `date` and variables
- Basic log management workflow
- Prepared foundation for scheduled automation with `cron`

## Reflection
Learned how to apply scripting to recurring file maintenance tasks. These scripts illustrate the fundamentals of backup and rotation mechanisms found in production systems.
