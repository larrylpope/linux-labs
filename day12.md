# Day 12 - Logic & Control Structures

## Scripts
- `check_user.sh` -- conditional greeting based on username
- `compare_numbers.sh` -- numeric comparison
- `file_check.sh` -- file type check
- `loop_example.sh` -- for/while loop examples

## Key Concepts
- Conditional syntax (`if`, `elif`, `else`)
- Comparison operators for numbers and strings
- File tests (`f`, `-d`, `-e`)
- Loops (`for`, `while`)
- Exit codes and conditional operators (`&&`, `||`)

## Reflection
Learned how logic and repetition enable scripts to make decisions and process files dynamically.
